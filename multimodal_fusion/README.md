# 多模态融合网络 - 特征融合模块

## 项目简介

本模块是基于SAR影像的电力线路损坏识别系统的特征融合部分，负责将光学图像和SAR图像的特征进行融合，生成融合后的伪光学图像，用于后续的目标检测任务。

## 目录结构

```
├── data/                  # 数据集目录（空文件夹结构）
│   ├── optical/           # 光学图像（直接放入）
│   ├── sar/               # SAR图像（直接放入）
│   └── fused/             # 融合后的图像（自动生成）
├── models/                # 模型权重（空文件夹）
├── multimodal_fusion.py   # 多模态融合网络
└── README.md              # 项目说明
```

## 技术特点

1. **多模态融合**：结合光学图像的纹理信息和SAR图像的结构信息
2. **注意力机制**：自动学习两种模态的重要性权重
3. **边缘保持**：保留电力线路的边缘信息，提高检测精度
4. **云雾穿透**：利用SAR不受云雾影响的特性，提高恶劣天气下的检测能力
5. **混合精度训练**：提高训练速度和显存利用率

## 环境要求

- Python 3.9+
- PyTorch 1.10+
- OpenCV
- NumPy
- PIL

## 使用方法

### 1. 准备数据

1. 将光学图像直接放入 `data/optical/` 目录
2. 将SAR图像直接放入 `data/sar/` 目录
3. 确保文件名对应，即相同名称的光学和SAR图像是配对的

### 2. 训练融合模型并生成融合图像

```bash
python multimodal_fusion.py
```

运行过程包括：
- 检查是否存在训练好的模型
- 如果模型不存在，训练融合网络
- 生成所有图像的融合结果并保存到 `data/fused/` 目录

## 模型架构

- **特征提取**：使用深度卷积网络提取光学和SAR的高维特征
- **注意力机制**：空间注意力和通道注意力，自动学习模态权重
- **特征融合**：通过卷积网络融合两种模态的特征
- **边缘保持**：保留电力线路的边缘信息
- **特征恢复**：通过渐进式上采样恢复到原始图像尺寸

## 训练参数

- **特征维度**：96
- **图像尺寸**：640×640
- **批量大小**：2
- **训练轮数**：100
- **损失函数**：L1 + 梯度 + SSIM
- **早停策略**：20轮无增益停止

## 输出

- 融合后的图像保存在 `data/fused/` 目录
- 训练好的模型保存在 `models/` 目录

## 注意事项

- 本模块只负责特征融合，不包含目标检测部分
- 实际数据集需要单独准备并直接放入 `data/optical/` 和 `data/sar/` 目录
- 模型权重较大，建议使用Git LFS管理
- 运行前确保所有目录结构已正确创建
